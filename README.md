# PivotTable CSV

## Project setup

É um projeto sem empacotamento, então basta abrir o index.html no navegador ou iniciar um http serve nos arquivos da raiz.

```
# http server Python 2
python -m SimpleHTTPServer

# http sever Python 3
python3 -m http.server
```
